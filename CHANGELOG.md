## v0.2.0 — 19-07-2026

Первый оформленный релиз.

- Демон `claude-pocketd`: API сессий, очередь в SQLite, SSE-стрим, usage/context, установка `curl | bash` + systemd
- Приложение: подключение по SSH (пароль/ключ), список сессий, вкладки, чат со стримом, свёрнутые действия агента, прерывание, слеш-команды, effort/режим прав
- Лимиты 5ч/7д и счётчик токенов контекста (`24.1k / 1M (2%)`) в футере чата
- TOFU-проверка отпечатка сервера; автогенерация ключа устройства при первом входе по паролю
- Автопроверка обновлений по GitHub Releases (раз в сутки) со скачиванием и установкой из приложения
