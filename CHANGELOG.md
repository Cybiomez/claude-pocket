# Changelog

Формат: [semver](https://semver.org). Версия приложения берётся из git-тега (`v0.2.0` → `0.2.0`).

## Не выпущено

- Экран сессий: нижний футер с лимитами больше не перекрывается плавающей
  кнопкой добавления сессии (отступ справа)
- Экран сессий: заголовок вынесен над рядом кнопок (не смещается от их числа),
  карточки сессий скруглены как пузыри чата, индикатор загрузки и лимиты убраны
  в нижний футер; иконка выхода к выбору сервера отзеркалена (стрелка влево)
- Браузер файлов: кнопка-список стоит рядом с «Назад» — сразу возврат в список
  сессий из любой глубины каталогов, без многократного «назад»
- Кнопки панели ввода в чате масштабируются под ширину экрана (эталон 360dp,
  с ограничением 0.9–1.4×) — на крупных экранах не остаются мелкими
- Экран подключений: заголовок «Claude Pocket» и подпись вынесены над рядом
  кнопок — текст больше не ужимается к краю экрана
- Меню настроек хода в чате: уровень усилий (effort) — ползунком с засечками
  и подписью текущего уровня над ним; у выбранного режима прав — галочка и подсветка
- Панель ввода: кнопки прикрепления, «/» и режима стали компактными квадратными
  (28dp с иконкой 16dp) — убран навязанный тач-таргет 48dp, больше не наезжают
- Установка обновления через системный установщик (как из браузера) — кнопка «Установить»
  теперь надёжно открывает установку; самообновление через PackageInstaller убрано
- Сравнение версий учитывает pre-release: финальный релиз корректно предлагается
  поверх своей dev-версии (0.4.5 поверх 0.4.5-dev.2), между dev-сборками — по суффиксу

## v0.4.5 — 19-07-2026

- Браузер файлов: в корне доступной области (домашней папке) кнопка «назад»
  возвращает в список сессий вместо ошибки 403 (попытка уйти выше корня)
- Экран сессий: кнопка выхода к выбору сервера и кнопка смены темы
- Обновление в два шага: «Обновить» скачивает APK, затем кнопка становится
  «Установить» и запускает уже скачанный файл (можно нажать повторно, если
  системный установщик замял установку)
- Починен тост о смене канала обновлений — больше не перебивается проверкой

## v0.4.0 — 19-07-2026

- Прикрепление файлов к сообщению: кнопка-скрепка, выбор через системный
  диалог, чипы прикреплённых файлов с удалением. Картинками в диалог уходят
  только форматы, которые принимает API (jpeg/png/gif/webp), остальное —
  файлом с путём на сервере; ограничение 8 МБ
- Просмотр файлов сервера (read-only): кнопка на экране сессий открывает браузер
  файлов — навигация по каталогам, текст с рендером markdown, код моноширинно,
  картинки; предел файла 2 МБ (ограничение демона)
- Стрелка «назад к списку» слева сверху в форме подключения (нового и редактирования)
- Переключатель канала обновлений на экране подключений (значок колбы): latest —
  только стабильные релизы, dev — плюс тестовые pre-release сборки
- Пузыри чата заметно скруглены с обеих сторон (маленький «хвостик» у своего угла);
  ответы ассистента на чуть более тёмном фоне — сообщения различимее
- Светлая тема в каноничных цветах (бежевый фон, тёмно-серый текст, терракотовый
  акцент) и переключатель темы на экране подключений: система / светлая / тёмная.
  Все цвета вынесены в единую палитру (ui/Theme.kt)
- Панель ввода в чате: кнопки «/» и режима меньше (окантовки не пересекаются),
  все кнопки выровнены по центру строки поля ввода
- Поддержка pre-release-версий в сборке и CI: теги с суффиксом (`v0.4.0-dev.1`)
  публикуются как pre-release, суффикс идёт в имя версии, versionCode по базовой

## v0.3.5 — 19-07-2026

- Кнопка-ссылка на GitHub-репозиторий (значок GitHub) в шапке экрана подключений
- Кнопка команд в чате: символ «/» вместо решётки; у кнопок «/» и режима — лёгкая окантовка
- Результат проверки обновлений — всплывающим сообщением внизу экрана («Установлена последняя
  версия», без обращений), вместо строки в интерфейсе
- Установка обновлений через PackageInstaller (самообновление): первое обновление — с одним
  системным подтверждением, дальше на Android 12+ без диалогов; ACTION_VIEW остался запасным путём
- Проверка, что APK скачался целиком, с понятной ошибкой вместо «приложение не установлено»
- Тексты интерфейса переведены на безличные формулировки (без обращений «на ты»)
- Исправлено прописывание ключа устройства: JSch не умеет сохранять приватный ed25519,
  из-за чего ключ молча не создавался и вход оставался по паролю. Теперь ключ — RSA 3072
  (цикл проверен вживую), а неудача прописывания показывается сообщением, а не глотается

## v0.3.0 — 19-07-2026

- Экран входа: список сохранённых подключений (несколько серверов), кнопка «Новое подключение»
- Карточки подключений: удобное имя, значок способа входа, карандаш — редактирование
  (имя, данные, удаление с подтверждением), перетаскивание за карточку — свой порядок,
  порядок запоминается
- Галочка «Запомнить подключение»: без неё — разовый вход без сохранения и без прописывания
  ключа устройства на сервер
- Вход по ключу устройства теперь в приоритете над паролем (`PreferredAuthentications`)
- Список подключений шифруется (AES-GCM, ключ в Android Keystore), бэкап данных приложения
  запрещён — пароли и ключи не уходят в облачный бэкап
- Кнопка «Проверить обновления» в шапке экранов подключений и сессий, с явным ответом
  «у тебя последняя версия» / ошибка сети

## v0.2.0 — 19-07-2026

Первый оформленный релиз.

- Демон `claude-pocketd`: API сессий, очередь в SQLite, SSE-стрим, usage/context, установка `curl | bash` + systemd
- Приложение: подключение по SSH (пароль/ключ), список сессий, вкладки, чат со стримом, свёрнутые действия агента, прерывание, слеш-команды, effort/режим прав
- Лимиты 5ч/7д и счётчик токенов контекста (`24.1k / 1M (2%)`) в футере чата
- TOFU-проверка отпечатка сервера; автогенерация ключа устройства при первом входе по паролю
- Автопроверка обновлений по GitHub Releases (раз в сутки) со скачиванием и установкой из приложения
